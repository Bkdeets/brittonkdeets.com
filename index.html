Price Pattern Search

As my course project in Design and Analysis of Algorithms, I wrote a program in Java to search for patterns in historical stock prices. I took a segment of daily stock prices of length n and converted the value of the price to 1 or 0 depending on if the price changed positively or negatively compared to the previous day. If a certain pattern of 1's and 0's resulted in an n+1 price value of 1, then I would consider that pattern an indication that the price would go up after that pattern. I also took into account the size of the price change and weighted the resulting expected return value based on that change. From this I developed a backtest trading long if there is a positive expected return and short if for a certain pattern there showed negative epected return.


On average the strategy tied buy and hold investing for the same period, but with more sophisticated entry and exit strategies, I think this simple strategy could be more profitable and consistent.
